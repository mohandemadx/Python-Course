# Python Creators Lab  
**Code → Screen → Real World**

## Course Overview
- **Duration:** 10 Weeks  
- **Sessions:** 2 per week (20 total)  
- **Age Group:** 14–16  
- **Goal:** Learn Python through games, visuals, and real-world projects  

---

## Teaching Philosophy
- Learn by building  
- Every lesson creates something visible  
- Encourage creativity and experimentation  

---
